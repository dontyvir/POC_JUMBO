Provee las clases e interfaces para la adminsitración de las conexiones a la base de datos.

Las configuraciones de la conexión se encuentran en el archivo de properties conexion.properties y son las siguientes:

  • conf.db.tipo: tipo de conexión las opciones son datasource o drivemanager
  • conf.db.datasource.lookup: nombre del jndi para la conexión cuando la conexión es vía datasource
  • conf.db.drivemanager.classforname: classforname cuando la conexión es vía drivemanager
  • conf.db.drivemanager.URL: URL de conexión cuando la conexión es vía drivemanager
  • conf.db.drivemanager.pass: clave cuando la conexión es vía drivemanager
  • conf.db.drivemanager.usr: nombre de usuario cuando la conexión es vía drivemanager
