Provee las clases e interfaces para la conexión al repositorio de datos.

Las clases involucradas son:

  • EmpresasDAO: Interfaz en la cual se publican los métodos que deben ser implementados en cualquier tipo de conexión.
  • SucursalesDAO: Interfaz en la cual se publican los métodos que deben ser implementados en cualquier tipo de conexión.
  • DireccionesDAO: Interfaz en la cual se publican los métodos que deben ser implementados en cualquier tipo de conexión.
  • CompradoresDAO: Interfaz en la cual se publican los métodos que deben ser implementados en cualquier tipo de conexión.
  • DAOFactory: Clase abstracta que permite la selección de los diferentes tipos de conexión al repositorio de datos.
  • HibernateDAOFactory: Crea una conexión al repositorio utilizando Hibernate.
  • JdbcDAOFactory: Crea una conexión al repositorio utilizando JDBC
  • HibernateEmpresasDAO: Clase que contiene los diferentes métodos para interactuar con la base de datos.
  • JdbcEmpresasDAO: Clase que contiene los diferentes métodos para interactuar con la base de datos.
